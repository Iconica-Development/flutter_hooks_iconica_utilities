## Getting started

Add flutter_hooks_iconica_utilities to your pubspec alongside flutter_hooks

```yml
dependencies:
  flutter_hooks_iconica_utilities:
    git:
      url: https://github.com/Iconica-Development/flutter_hooks_iconica_utilities
      ref: 1.0.0
  flutter_hooks: ^0.20.0
```
